# Sistem Manajemen Kontak Sederhana

# Deksripsi Singkat 
Sistem Manajemen Kontak berbasis web yang ringan dan modern. Sistem ini tidak menggunakan database; sebagai gantinya, semua data kontak (tambah, edit, hapus) disimpan sementara menggunakan PHP Session.

data kontak akan tersimpan selama Anda tidak menutup browser atau me-restart server. Aplikasi ini dirancang dengan antarmuka yang bersih menggunakan Tailwind CSS, fungsionalitas pencarian, dan logika backend yang dipecah per fitur untuk memudahkan pemeliharaan.

# Fitur :
ini adalah fitur-fitur utamanya secara singkat:

Tambah, Lihat, Edit, dan Hapus Kontak (CRUD).
Penyimpanan data sementara menggunakan PHP Session (tanpa database).
Formulir dinamis yang berubah fungsi dan warna tombol saat mode "Edit".
Fitur pencarian kontak.
Notifikasi (pesan) untuk umpan balik (berhasil, gagal, error).
Validasi input (nama, email, telepon) di sisi server.
